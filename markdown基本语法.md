下面是 **Markdown 基本语法速查表**（常用且够用的那一套）👇

---

## 1️⃣ 标题

```md
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

效果：

# 一级标题

## 二级标题

### 三级标题

---

## 2️⃣ 段落与换行

- **空一行**表示新段落
    
- 行尾加两个空格再回车 → 强制换行
    

```md
第一行  
第二行
```

---

## 3️⃣ 加粗 / 斜体 / 删除线

```md
**加粗**
*斜体*
***加粗+斜体***
~~删除线~~
```

效果：  
**加粗**  
_斜体_  
_**加粗+斜体**_  
~~删除线~~

---

## 4️⃣ 列表

### 无序列表

```md
- 苹果
- 香蕉
  - 小香蕉
```

### 有序列表

```md
1. 第一项
2. 第二项
```

---

## 5️⃣ 引用

```md
> 这是引用
>> 二级引用
```

效果：

> 这是引用
> 
> > 二级引用

---

## 6️⃣ 代码

### 行内代码

```md
`print("Hello")`
```

效果：`print("Hello")`

### 代码块

````
```python
def hello():
    print("Hello Markdown")
```
````

---

## 7️⃣ 链接

```md
[OpenAI](https://openai.com)
```

效果：  
[OpenAI](https://openai.com/)

---

## 8️⃣ 图片

```md
![图片描述](图片链接)
```

示例：

```md
![logo](https://example.com/logo.png)
```

---

## 9️⃣ 分割线

```md
---
```

## 效果：

## 🔟 表格

```md
| 姓名 | 年龄 | 职业 |
| ---- | ---- | ---- |
| 张三 | 20 | 学生 |
| 李四 | 30 | 工程师 |
```

|姓名|年龄|职业|
|---|---|---|
|张三|20|学生|
|李四|30|工程师|
